# Finances & Cash Logs

## Overview
Tracks the weekly cash (Kas) payments for division members.

## Operations
1.  **Automated Billing**: Every division member is automatically billed weekly for cash based on the global configuration (`app:generate-weekly-cash`).
2.  **Payment Processing**: When a member pays, Secretary/Admin updates the `CashLogs` status to `Paid`.
3.  **Debt Tracking**: The system tracks who has outstanding arrears.
